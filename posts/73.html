Undo Manager

March 27, 2016 anmol-lf

NSUndoManager is a powerful feature of cocoa which lets users undo and redo their actions. It maintains two stacks – undo and redo stacks. Each time a registered event is triggered, a selector with arguments to undo that event is added to the undo stack. Calling undo/redo just shifts the selector from one stack to another.

class ViewController: UIViewController {
    
    @IBOutlet var label: UILabel!
    @IBOutlet var textField: UITextField!
    
    let undoManagerForLabel = NSUndoManager()
    
    func updateText(text: String?) {
        self.undoManagerForLabel.registerUndoWithTarget(self,
            selector: Selector("updateText:"),
            object: self.label.text)
        self.textField.text = text
        self.label.text = text
    }
    
    @IBAction func okay(sender: UIButton) {
        self.updateText(self.textField.text)
    }
    
    @IBAction func undo(sender: UIButton) {
        self.undoManagerForLabel.undo()
    }
    
    @IBAction func redo(sender: UIButton) {
        self.undoManagerForLabel.redo()
    }
 
}

The above code updates the label whenever okay action is called and the undo and redo actions are used to undo and redo the label text.
