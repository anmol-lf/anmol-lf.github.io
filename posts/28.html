Function Decorator

March 25, 2016 anmol-lf

Swift borrows a lot of features from the functional family. Most of the traits of a functional language is available in swift. Functions are treated as first class citizens in Swift and are value typed. Function decoration a way to extend functionality of a function.

var aFunction = {
    print("a")
}
 
print("Before decorating: ")
aFunction()
 
var tempFunction = aFunction
aFunction = {
    print("b")
    tempFunction()
}
 
print("After decorating:")
aFunction()

Here, aFunction was extended to print ‘b’ first and then do what aFunction is expected to do.
